## Platform Capabilities

### Scaling Metrics

| Metric | Value | Description |
|--------|-------|-------------|
| **Cold Start** | <10ms | WASM agent instantiation |
| **Agents/Node** | 10,000+ | SpinKube density |
| **Inference Latency** | <50ms | Local GPU (Mac Silicon) |
| **Cross-Cluster RTT** | <100ms | Federation mesh |
| **Auto-Scale Time** | <30s | HPA response |
| **Self-Heal Recovery** | <60s | Automatic pod replacement |
| **Memory/Agent** | ~2MB | WASM isolation overhead |

### Deployment Modes

1. **Single Node**: Mac Studio / MacBook Pro
2. **Edge Cluster**: K3s on Raspberry Pi + Intel NUC
3. **Hybrid Cloud**: Local + AWS/GCP burst
4. **Full Federation**: Multi-region Gaia Network

## Quick Start

```bash
# 1. Deploy the platform
./platform/scripts/deploy-hyperscale.sh

# 2. Start agents
./platform/scripts/launch-swarm.sh --agents 1000 --type coder

# 3. Scale to cloud (optional)
./platform/scripts/cloud-burst.sh --provider aws --region us-west-2
```
